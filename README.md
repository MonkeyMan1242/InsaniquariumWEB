# InsaniquariumWEB

![Large Guppy](https://i.imgur.com/hT1mjQV.png)

Adds 20+ Insaniquarium fish and aliens to WEBFISHING! Can you catch/defeat the insanely rare Cyrax? Fish are found in lakes and rivers, and aliens in the ocean.
